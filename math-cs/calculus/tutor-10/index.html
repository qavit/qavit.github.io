A. 基本積分公式 ^A

A.1 多項式函數的積分 ^poly

定理： 若 \(n \neq -1\)，則

\[ \int x^n \dd{x} = \frac{x^{n+1}}{n+1} + C \]

其中，\(C\) 是積分常數。

證明： 根據微積分基本定理，我們知道微分和積分是互為反操作的。考慮函數 \(F(x) = \frac{x^{n+1}}{n+1}\)，則

\[ F'(x) = \dv{x} \left( \frac{x^{n+1}}{n+1} \right) = x^n \]

因此，

\[ \int x^n \dd{x} = \frac{x^{n+1}}{n+1} + C \]

例題： 計算 \(\int x^3 \dd{x}\)。

解答：

\[ \int x^3 \dd{x} = \frac{x^{3+1}}{3+1} + C = \frac{x^4}{4} + C \]

A.2 指數函數的積分 ^exp

定理：

\[ \int e^x \dd{x} = e^x + C \]

證明： 根據微積分基本定理，我們知道函數 \(F(x) = e^x\) 的導數是 \(F'(x) = e^x\)，因此

\[ \int e^x \dd{x} = e^x + C \]

例題： 計算 \(\int e^{2x} \dd{x}\)。

解答： 設 \(u = 2x\)，則 \(du = 2 \dd{x}\)，即 \(dx = \frac{1}{2} du\)。

\[ \int e^{2x} \dd{x} = \int e^u \cdot \frac{1}{2} \dd{u} = \frac{1}{2} \int e^u \dd{u} = \frac{1}{2} e^u + C = \frac{1}{2} e^{2x} + C \]

A.3 對數函數的積分 ^ln

定理：

\[ \int \frac{1}{x} \dd{x} = \ln |x| + C \]

證明： 根據微積分基本定理，我們知道函數 \(F(x) = \ln |x|\) 的導數是 \(F'(x) = \frac{1}{x}\)，因此

\[ \int \frac{1}{x} \dd{x} = \ln |x| + C \]

例題： 計算 \(\int \frac{2}{x} \dd{x}\)。

解答：

\[ \int \frac{2}{x} \dd{x} = 2 \int \frac{1}{x} \dd{x} = 2 \ln |x| + C \]

A.4 三角函數的積分 ^tri

定理：

\[ \int \sin x \dd{x} = -\cos x + C \] \[ \int \cos x \dd{x} = \sin x + C \]

證明： 根據微積分基本定理，我們知道 \(\frac{d}{dx} (-\cos x) = \sin x\) 和 \(\frac{d}{dx} (\sin x) = \cos x\)，因此

\[ \int \sin x \dd{x} = -\cos x + C \] \[ \int \cos x \dd{x} = \sin x + C \]

例題： 計算 \(\int \cos(3x) \dd{x}\)。

解答： 設 \(u = 3x\)，則 \(du = 3 \dd{x}\)，即 \(dx = \frac{1}{3} du\)。

\[ \int \cos(3x) \dd{x} = \int \cos u \cdot \frac{1}{3} \dd{u} = \frac{1}{3} \int \cos u \dd{u} = \frac{1}{3} \sin u + C = \frac{1}{3} \sin(3x) + C \]

B. 變數變換法 ^B

怎麼求 \(\displaystyle\int 2x\sqrt{1+x^2}\dd{x}\)？ 提示：令 \(u=1+x^2\)

【定理】變數變換法則

如果 \(u=g(x)\) 是一個可微函數，其值域為區間 \(I\)，且函數 \(f\) 在 \(I\) 上連續，則

\[\boxed{\int f(g(x))g'(x)\dd{x}=\int f(u)\dd{u}}\]

%% (§5.5, Theorem 4, p.420) %%

例題：

  1. 計算 \(\displaystyle\int{x^3\cos{\left(x^4+2\right)}\dd{x}}\)。%% (§5.5, Example 1, p.420) %%
  2. 計算 \(\displaystyle\int{\sqrt{2x+1}\dd{x}}\)。%% (§5.5, Example 2, p.421) %%
  3. 計算 \(\displaystyle\int{\frac{x}{\sqrt{1-4x^2}}\dd{x}}\)。%% (§5.5, Example 3, p.421) %%

【定理】定積分的變數變換法則

如果 \(g:[a,b]\to\mathbb{R}\) 是一個可微函數（且 \(u=g(x)\)），且 \(g'\) 在 \([a,b]\) 上連續，且函數 \(f\) 在 \(g([a,b])\) 上連續，則

\[\boxed{\int^b_a f(g(x))g'(x)\dd{x}=\int^{g(b)}_{g(a)} f(u)\dd{u}}\]

%% (§5.5, Theorem 6, p.423) %%

例題：

  1. 計算 \(\displaystyle\int^4_0{\sqrt{2x+1}\dd{x}}\)。%% (§5.5, Example 7, p.423) %%
  2. 計算 \(\displaystyle \int^2_1 \frac{\dd{x}}{(3-5x)^2}\)。%% (§5.5, Example 8, p.424) %%
  3. 計算 \(\displaystyle \int^e_1 \frac{\ln{x}}{x}\dd{x}\)。%% (§5.5, Example 9, p.424) %%

【定理】對稱函數的積分

假設 \(f\) 在 \([-a,a]\) 上連續。

  • (a) 若 \(f\) 是偶函數，則 \(\displaystyle\int^a_{-a}{f(x)\dd{x}} = 2\int^a_0{f(x)\dd{x}}\)。
  • (a) 若 \(f\) 是奇函數，則 \(\displaystyle\int^a_{-a}{f(x)\dd{x}} = 0\)。

例題：

  1. 計算 \(\displaystyle \int^1_{-1} \frac{\tan{x}}{1+x^2+x^4}\dd{x}\)。%% (§5.5, Example 11, p.425) %%

C. 分部積分法 (Integration by Parts) ^C

設 \(f(x)\)、\(g(x)\) 是可微函數，導數的積法則

\[\dv{x}[f(x)g(x)]=f'(x)g(x)+f(x)g'(x)\]

由 FTC

\[f(x)g(x)=\int{\big[f'(x)g(x)+f(x)g'(x)\big]\dd{x}}\] \[\boxed{\int{f(x)g'(x)\dd{x}}=f(x)g(x)-\int{f'(x)g(x)\dd{x}}}\]

此為分部積分（integration by parts）的公式。令 \(u=f(x)\)、\(v=g(x)\)，則 \(\dd{u}=f'(x)\dd{u}\) 、\(\dd{v}=g'(x)\dd{v}\)，於是

\[ \int u \dd{v} = uv - \int v \dd{u} \]

其中，\(u\) 和 \(v\) 是可微函數。

例題：

  1. 計算 \(\int x e^x \dd{x}\)。

    解答： 設 \(u = x\)，\(dv = e^x \dd{x}\)，則 \(du = dx\)，\(v = e^x\)。

    \[ \int x e^x \dd{x} = x e^x - \int e^x \dd{x} = x e^x - e^x + C = e^x (x - 1) + C \]

  2. 計算 \(\int x \sin{x} \dd{x}\)。%% (§7.1, Example 1, p.486) %%

  3. 計算 \(\int{\ln{x}\dd{x}}\)。%% (§7.1, Example 2, p.487) %%

  4. 計算 \(\int{e^x\sin{x}\dd{x}}\)。%% (§7.1, Example 3, p.489) %%

C.1 定積分的分部積分 ^C-1

\[\boxed{\int^b_a{f(x)g'(x)\dd{x}}=f(x)g(x)\Big|^b_a-\int^b_a{f'(x)g(x)\dd{x}}}\]
