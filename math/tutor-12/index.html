3. 雙曲代換法 ^3

雙曲代換法（hyperbolic substitution）是類似於三角代換的一種技巧，常用於含有 \(\sqrt{a^2 \pm x^2}\) 或 \(\sqrt{x^2 \pm a^2}\) 類型的積分。它利用雙曲函數的特性，將複雜的被積函數轉化為簡單的代數表達式。

3.1 雙曲函數 ^3-1

3.1.1 雙曲函數的定義 ^3-1-1

  • 雙曲正弦函數（sinh）：\[\sinh{x} = \frac{e^x - e^{-x}}{2}\]
  • 雙曲餘弦函數（cosh）：\[\cosh{x} = \frac{e^x + e^{-x}}{2}\]
  • 雙曲正切函數（tanh）：\[\tanh{x} = \frac{\sinh{x}}{\cosh{x}} = \frac{e^x - e^{-x}}{e^x + e^{-x}}\]
  • 雙曲餘切函數（sech）：\[\coth{x} = \frac{\cosh{x}}{\sinh{x}} = \frac{e^x + e^{-x}}{e^x - e^{-x}}, \quad x \neq 0\]
  • 雙曲正割函數（sech）：\[\operatorname{sech}{x} = \frac{1}{\cosh{x}} = \frac{2}{e^x + e^{-x}}\]
  • 雙曲餘割函數（csch）：\[\operatorname{csch}{x} = \frac{1}{\sinh{x}} = \frac{2}{e^x - e^{-x}}, \quad x \neq 0\]

雙曲函數也可以從複數來定義，詳見：

  1. 維基百科 - 雙曲函數
  2. Wikipedia - Hyperbolic functions
  3. Wolfram Mathworld - Hyperbolic functions
  4. Review of the Hyperbolic Functions

3.3.2 反雙曲函數的定義 ^3-3-2

  • 反雙曲正弦函數（arsinh）：\[\operatorname{arsinh}{x} =\sinh^{-1}{x}= \ln(x + \sqrt{x^2 + 1})\]
  • 反雙曲餘弦函數（arcosh）：\[\operatorname{arcosh}{x} =\cosh^{-1}{x} = \ln(x + \sqrt{x^2 - 1}), \quad x \geq 1\]
  • 反雙曲正切函數（artanh）**：\[\operatorname{tanh}{x} =\tanh^{-1}{x} = \frac{1}{2} \ln\left(\frac{1 + x}{1 - x}\right), \quad |x| < 1\]
  • 反雙曲餘切函數（arcoth）：\[\operatorname{arcoth}{x} =\coth^{-1}{x} = \frac{1}{2} \ln\left(\frac{x + 1}{x - 1}\right), \quad |x| > 1\]
  • 反雙曲正割函數（arsech）：\[\operatorname{arsech}{x}=\operatorname{sech}^{-1}{x} = \ln\left(\frac{1 + \sqrt{1 - x^2}}{x}\right), \quad 0 < x \leq 1\]
  • 反雙曲餘割函數（arcsch）：\[\operatorname{arcsch}{x}=\operatorname{csch}^{-1}{x} = \ln\left(\frac{1 + \sqrt{1 + x^2}}{x}\right), \quad x \neq 0\]

3.3.3 雙曲恆等式 ^3-3-3

以下僅列出最常見的（還有後面計算會用到的）雙曲恆等式（hyperbolic identities），

  • 平方差恆等式 1：\[\cosh^2{x} - \sinh^2{x} = 1\]
  • 平方差恆等式 2：\[\tanh^2{x} - \coth^2{x} = 1\]
  • 倍角公式 \[\begin{aligned}\sinh{2x}&=2\sinh{x}\cosh{x},\\\cosh{2x}&=\cosh^2{x}+\sinh^2{x}\\&=2\cosh^2{x}-1\\&=2\sinh^2{x}+1\\\end{aligned}\]^double
  • 半角公式 \[\begin{aligned}\sinh^2{\frac{x}{2}}&=\frac{\cosh{x}+1}{2}\\\cosh^2{\frac{x}{2}}&=\frac{\cosh{x}-1}{2}\end{aligned}\]^half

完整可以參考

  1. 維基百科 - 雙曲函數恆等式
  2. Wikipedia - Hyperbolic functions#Useful relations
  3. Hyperbolic Function Identities

3.1.4 雙曲函數的導函數 ^3-1-4

  • \(\displaystyle \dv{x}{\sinh{x}}=\cosh{x}\)
  • \(\displaystyle \dv{x}{\cosh{x}}=\sinh{x}\)
  • \(\displaystyle \dv{x}{\tanh{x}}=1-\tanh^2{x}=\operatorname{sech}^2{x}=\frac{1}{\cosh^2{x}}\)

3.3 例題 ^3-3

例題 3-1 ^eq3-1

證明

\[\int{\sqrt{x^2 + a^2} \dd{x}}=\frac{1}{2}\left[x\sqrt{x^2 + a^2} + a^2 \sinh^{-1}{\left(\frac{x}{a}\right)} \right] + C.\]

證.

  1. 雙曲代換： 令 \(x = a \sinh{t}\)，則 \(\dd{x} = a \cosh{t} \dd{t}\)，且

    \[ \sqrt{x^2 + a^2} = \sqrt{a^2 \sinh^2{t} + a^2} = a \sqrt{\sinh^2{t} + 1} = a \cosh{t}.\]

  2. 積分變換：

    \[\int \sqrt{x^2 + a^2} \dd{x} = \int a \cosh{t} \cdot a \cosh{t} \dd{t} = a^2 \int \cosh^2{t} \dd{t}\]

  3. 化簡積分： 利用[[#^half|雙曲半角恆等式]] \(\cosh^2{t} = \frac{1}{2}(1 + \cosh{2t})\)，得

    \[ a^2 \int \cosh^2{t} \dd{t} = a^2 \int \frac{1 + \cosh{2t}}{2} \dd{t} = \frac{a^2}{2} \int (1 + \cosh{2t}) \dd{t} \] \[ = \frac{a^2}{2} \left( t + \frac{1}{2} \sinh{2t} \right) + C \]

  4. 還原變數： 由於 \(x = a \sinh{t}\)，因此 \(t = \sinh^{-1}{\left(\frac{x}{a}\right)}\) ，然後利用[[#^double|雙曲倍角恆等式]] \(\sinh{2t} = 2 \sinh{t} \cosh{t}\)：

    \[ \begin{aligned} \int{\sqrt{x^2 + a^2} \dd{x}}&= \frac{a^2}{2} \left( \sinh^{-1}{\left(\frac{x}{a}\right)} + \frac{1}{2} \cdot 2 \sinh{t} \cosh{t} \right) + C\\ &= \frac{a^2}{2} \left( \sinh^{-1}{\left(\frac{x}{a}\right)} + \frac{x \sqrt{x^2 + a^2}}{a^2} \right) + C\\ \end{aligned} \]

  5. 與[[tutor-11#^eq2-6-ans|例題 2-6]] 比較

    \[ \begin{aligned} \int \sqrt{x^2 + a^2} \dd{x} &= \frac{a^2}{2} \left( \frac{x \sqrt{x^2 + a^2}}{a^2} + \ln{|\sqrt{x^2 + a^2} + x|} \right) + C\\ &= \frac{a^2}{2} \left(\frac{x \sqrt{x^2 + a^2}}{a^2} + \sinh^{-1}{\left(\frac{x}{a}\right)} \right) + C \end{aligned} \]

4. 部分分式法 ^4

4.1 有理函數 ^4-1

有理函數（rational function）是兩個多項式函數的比值，形式為：

\[ f(x) = \frac{P(x)}{Q(x)} \]

其中，\(P(x)\) 和 \(Q(x)\) 為多項式函數，且 \(Q(x) \neq 0\)。

4.2 部分分式分解 ^4-2

對於有理函數 \(f(x) = \frac{P(x)}{Q(x)}\)，且 \(\deg(P) < \deg(Q)\)，可以將 \(f(x)\) 分解為部分分式（partial fractions）的和

\[ \frac{P(x)}{Q(x)} = \sum_{i} \frac{A_i}{(x - \alpha_i)^{k_i}} + \sum_{j} \frac{B_j x + C_j}{(x^2 + d_j x + e_j)^{m_j}}。 \]

部分分式積分法（integration by partial fractions）就是將有理函數表示為若干部分分式的和，再對每一部分分式進行積分的方法。

根據分母 \(Q(x)\) 的不同形式，可以分為以下四種情況。

情況 I：\(Q(x)\) 是不同線性因子的乘積 ^part-case1

當 \(Q(x)\) 是不同線性因子的乘積時，可以將 \(f(x)\) 分解為以下形式：

\[ \frac{P(x)}{Q(x)} = \frac{P(x)}{(x-a_1)(x-a_2)\cdots(x-a_n)} = \frac{A_1}{x-a_1} + \frac{A_2}{x-a_2} + \cdots + \frac{A_n}{x-a_n} \]

其中，\(A_i\) 是常數，通過解聯立方程可以求出。

情況 II：\(Q(x)\) 是線性因子的乘積，其中某些因子重複 ^part-case2

當 \(Q(x)\) 是線性因子的乘積，且某些因子重複時，可以將 \(f(x)\) 分解為以下形式：

\[ \frac{P(x)}{Q(x)} = \frac{P(x)}{(x-a_1)^k (x-a_2)\cdots(x-a_n)} = \frac{A_1}{x-a_1} + \frac{A_2}{(x-a_1)^2} + \cdots + \frac{A_k}{(x-a_1)^k} + \frac{B_2}{x-a_2} + \cdots + \frac{B_n}{x-a_n} \]

其中，\(A_i\) 和 \(B_i\) 是常數，通過解聯立方程可以求出。

情況 III：\(Q(x)\) 包含不可約二次因子，且每個因子不重複 ^part-case3

當 \(Q(x)\) 包含不可約二次因子，且每個因子不重複時，可以將 \(f(x)\) 分解為以下形式：

\[ \frac{P(x)}{Q(x)} = \frac{P(x)}{(x^2+bx+c)(x-a_1)\cdots(x-a_n)} = \frac{Ax+B}{x^2+bx+c} + \frac{C_1}{x-a_1} + \cdots + \frac{C_n}{x-a_n} \]

其中，\(A\), \(B\) 和 \(C_i\) 是常數，通過解聯立方程可以求出。

情況 IV：\(Q(x)\) 包含重複的不可約二次因子 ^part-case4

當 \(Q(x)\) 包含重複的不可約二次因子時，可以將 \(f(x)\) 分解為以下形式：

\[ \begin{align} \frac{P(x)}{Q(x)} &= \frac{P(x)}{(x^2+bx+c)^k (x-a_1)\cdots(x-a_n)} \\ &= \frac{A_1 x + B_1}{x^2 + bx + c} + \frac{A_2 x + B_2}{(x^2 + bx + c)^2} + \cdots + \frac{A_k x + B_k}{(x^2 + bx + c)^k} + \frac{C_1}{x-a_1} + \cdots + \frac{C_n}{x-a_n} \end{align} \]

其中，\(A_i\), \(B_i\) 和 \(C_i\) 是常數，通過解聯立方程可以求出。

4.3 例題 ^4-3

例題 4-1（情況 I） ^eq4-1

求 \(\displaystyle \int {\frac{3x + 5}{(x-1)(x+2)} \dd{x}}\)。

解.

  1. 先對被積函數做部分分式分解。令

    \[\frac{3x + 5}{(x-1)(x+2)} = \frac{A}{x-1} + \frac{B}{x+2}\]

    將兩邊的分母消去：

    \[3x + 5 = A(x+2) + B(x-1)\]

    選擇合適的 \(x\) 值求解 \(A\) 和 \(B\)：

    • 當 \(x = 1\) 時，\(3\cdot1 + 5 = A\cdot(1+2) + B\cdot0\)，\(A = \frac{8}{3}\)。
    • 當 \(x = -2\) 時，\(3\cdot(-2) + 5 = A\cdot0 + B\cdot(-2-1)\)，\(B = -\frac{1}{3}\)。

    所以：

    \[\frac{3x + 5}{(x-1)(x+2)} = \frac{8/3}{x-1} - \frac{1/3}{x+2}\]

  2. 積分：

    \[\int \frac{3x + 5}{(x-1)(x+2)} \dd{x} = \int \frac{8/3}{x-1} \dd{x} - \int \frac{1/3}{x+2} \dd{x} = \frac{8}{3} \ln{|x-1|} - \frac{1}{3}\ln{|x+2|} + C.\]
例題 4-2（情況 II） ^eq4-2

求 \(\displaystyle \int \frac{x^2 + 1}{(x-1)^2(x+2)} \dd{x}\)。

解.

  1. 部分分式分解

    \[\frac{x^2 + 1}{(x-1)^2(x+2)} = \frac{4}{9(x-1)} + \frac{2}{3(x-1)^2} + \frac{5}{9(x+2)}.\]

  2. 積分：

    \[\int \frac{x^2 + 1}{(x-1)^2(x+2)} \dd{x} = \frac{4}{9}\ln{|x-1|} - \frac{2}{3(x-1)} + \frac{5}{9}\ln{|x+2|} + C.\]

例題 4-3（情況 III） ^eq4-3

求 \(\displaystyle\int {\frac{2x + 3}{x^2 + 4} \dd{x}}\)。

解. 被積函數本身已經不可再簡化。所求積分等於

\[\int \frac{2x}{x^2 + 4} \dd{x} + \int \frac{3}{x^2 + 4} \dd{x};\]

第一部分：

\[\int \frac{2x}{x^2 + 4} \dd{x} = \ln{|x^2 + 4|} + C_1;\]

第二部分：

\[\int \frac{3}{x^2 + 4} \dd{x} = \frac{3}{2} \int \frac{1}{(x/2)^2 + 1} \dd{x} = \frac{3}{2} \tan^{-1}{\left(\frac{x}{2}\right)} + C_2;\]

合併：

\[ \int \frac{2x + 3}{x^2 + 4} \dd{x} = \ln{|x^2 + 4|} + \frac{3}{2} \tan^{-1}{\left(\frac{x}{2}\right)} + C. \]
例題 4-4（情況 III） ^eq4-4

計算 \(\displaystyle\int \frac{2x + 3}{(x - 1)(x^2 + 1)} \, \dd{x}\)。

解.
首先，將分式分解為部分分式：

\[ \frac{2x + 3}{(x - 1)(x^2 + 1)} = \frac{A}{x - 1} + \frac{Bx + C}{x^2 + 1} \]

通分後，得到：

\[ 2x + 3 = A(x^2 + 1) + (Bx + C)(x - 1) \]

展開並比較係數，得到方程組：

\[ 2x + 3 = Ax^2 + A + Bx^2 - Bx + Cx - C \]

整理後：

\[ 2x + 3 = (A + B)x^2 + (C - B)x + (A - C) \]

比較兩邊的係數：

\[ \begin{cases} A + B = 0\\ C - B = 2\\ A - C = 3 \end{cases} \]

解此方程組，得到 \(A = \frac{5}{2}, B = -\frac{5}{2}, C = -\frac{1}{2}\)，因此部分分式為：

\[ \frac{2x + 3}{(x - 1)(x^2 + 1)} = \frac{\frac{5}{2}}{x - 1} + \frac{-\frac{5}{2}x - \frac{1}{2}}{x^2 + 1} \]

積分變為：

\[ \begin{align} \int{\frac{2x + 3}{(x - 1)(x^2 + 1)}\,\dd{x}} &= \int\left(\frac{\frac{5}{2}}{x - 1} + \frac{-\frac{5}{2}x - \frac{1}{2}}{x^2 + 1}\right)\dd{x} \\ &= \frac{5}{2}\int{\frac{1}{x - 1} \, \dd{x}} - \frac{5}{2} \int{\frac{x}{x^2 + 1}\, \dd{x}} -\frac{1}{2} \int{\frac{3}{x^2 + 1} \, \dd{x}} \end{align} \]

每一項分別計算，就得到

\[ \int{\frac{2x + 3}{(x - 1)(x^2 + 1)}\,\dd{x}} = \frac{5}{2}\ln|x - 1| + \frac{5}{4} \ln\left(x^2 + 1\right) + \frac{3}{2} \tan^{-1}{x} + C \]
例題 4-5（情況 IV） ^eq4-5

求 \(\displaystyle\int{\frac{x^2 + 2x + 1}{\left(x^2+1\right)^2}\,\dd{x}}\)。

解.

  1. 部分分式分解

    \[\frac{x^2 + 2x + 1}{\left(x^2+1\right)^2}= \frac{1}{x^2+1} + \frac{2x}{\left(x^2+1\right)^2}.\]

  2. 積分：

    \[\int{\frac{x^2 + 2x + 1}{\left(x^2+1\right)^2}\,\dd{x}} = \int{\frac{\dd{x}}{x^2+1}} + \int{\frac{2x}{\left(x^2+1\right)^2}\dd{x}};\]

    第一部分：使用[[tutor-11#^eq2-1|例題 2-1]] 的結果

    \[\int{\frac{\dd{x}}{x^2+1}}=\arctan{x}+C_1;\]

    第二部分：使用變數變換法，得到

    \[\int{\frac{2x}{\left(x^2+1\right)^2}\dd{x}}=-\frac{1}{x^2+1}+C_2.\]

  3. 合併得到

    \[\displaystyle\int{\frac{x^2 + 2x + 1}{\left(x^2+1\right)^2} \,\dd{x}}=\arctan{x}-\frac{1}{x^2+1}+C.\]

5. 降次積分法 ^5

5.1 原理 ^5-1

設

\[I_n=\int f_n(x)\dd{x}\]

是==被積函數包含整數參數==（例如初等函數的冪次、或超越函數與任意次數的多項式的乘積）的積分。

若 \(I_n\) 存在遞迴關係（recurrence relation），也就是說，「如果 \(I_n=\int f_n(x)\dd{x}\) 可以用 \(I_m=\int f_m(x)\dd{x}\) 來表示（\(n>m\)），我們就可以重複執行這項動作，降低表達式中的整數參數，直到積分可以計算為止。這就是降次積分法（Integration by reduction formulae）。

有許多函數都可使用降次積分法，詳見 Tables of integral reduction formulas。

5.2 例題 ^5-2

例題 5-1 ^eq5-1

計算 \(\int{\cos^n{x}\dd{x}}\)。

解.

\[ \begin{align} \int{{\color{blue}\cos^{n-1}{(x)}}\,{\color{green}\cos{(x)}\dd{x}}} &= \int{\color{blue}\left(\cos^{n-1}{x}\right)}{\color{green}\,\dd{(\sin{x})}} \\ &= {\color{blue}\cos^{n-1}{(x)}}\,{\color{green}\sin{(x)}} - \int{{\color{green}\sin{(x)}}\,{\color{blue}\dd{\left(\cos^{n-1}{x}\right)}}} \\ &= {\color{blue}\cos^{n-1}{(x)}}\,{\color{green}\sin{(x)}}\,{\color{blue}+\,(n-1)}\int{{\color{blue}\cos^{n-2}{(x)}}\,{\color{green}\sin{(x)}}\,{\color{blue}\sin{(x)}\dd{x}}} \\ &= \cos^{n-1}{(x)}\sin{(x)} + (n-1)\int\cos^{n-2}{(x)}\left(1-\cos^2{x}\right)\dd{x}\\ &= \cos^{n-1}{(x)}\sin{(x)} + (n-1)\int\cos^{n-2}{(x)}\dd{x}-(n-1)\int\cos^n{x}\dd{x}\\ &= \cos^{n-1}{(x)}\sin{(x)} + (n-1)I_{n-2}-(n-1)I_n \end{align} \]

求解 \(I_n\)：

\[ \begin{align} I_n+(n-1)I_n &= \cos^{n-1}{(x)}\sin{(x)}+(n-1)I_{n-2} \\ nI_n &= \cos^{n-1}{(x)}\sin{(x)}+(n-1)I_{n-2} \\ I_n &= \frac{1}{n}\cos^{n-1}{(x)}\sin{(x)}+\frac{n-1}{n}I_{n-2} \end{align} \]

所以

\[\int{\cos^n{x}\dd{x}} = \frac{1}{n}\cos^{n-1}{x}\sin{x}+\frac{n-1}{n}\int{\cos^{n-2}{x}\dd{x}}\]

舉例來說， \(n=5\) 時，

\[\int{\cos^5{x}\dd{x}} = \frac{1}{5}\cos^{4}{x}\sin{x}+\frac{4}{5}\int{\cos^{3}{x}\dd{x}}\] \[\int{\cos^3{x}\dd{x}} = \frac{1}{3}\cos^{2}{x}\sin{x}+\frac{2}{3}\int{\cos{x}\dd{x}}\]

所以

\[ \int{\cos^5{x}\dd{x}} = \frac{1}{5}\cos^{4}{x}\sin{x}+\frac{4}{5}\left(\frac{1}{3}\cos^{2}{x}\sin{x}+\frac{2}{3}\sin{x}\right)+C \]

例題 5-2 ^eq5-2

計算 \(\int x^n e^{ax} \dd{x}\)。

解.

例題 5-3 ^eq5-3

計算 \(\int{\sin^n{x}\dd{x}}\)。 %% §7.1, Example 6, p.489 %%

6. 其他積分技巧 (1) ^6

6.1 正切半角代換 ^6-1

正切半角代換（Tangent half-angle substitution），有時稱為 Weierstrass 代換（Weierstrass substitution），是將變數 \(t\) 替換為 \(t = \tan{\frac{\theta}{2}}\)，用以簡化含有三角函數的積分。

%% https://www.youtube.com/watch?v=u9KGuzWpUHw&list=PLP1Ynr8cs97uFPNReAr5Z4DT9VtvYCX1R&index=5 %%

代換規則

設 \(t = \tan{\frac{\theta}{2}}\)，則：

\[ \sin{\theta} = \frac{2t}{1 + t^2}，\quad \cos{\theta} = \frac{1 - t^2}{1 + t^2}，\quad \dd{\theta} = \frac{2 \, \dd{t}}{1 + t^2}。 \] \[ \sin{\frac{\theta}{2}}=\frac{t}{\sqrt{1+t^2}},\quad\cos{\frac{\theta}{2}}=\frac{1}{\sqrt{1+t^2}} \]

推導. 由二倍角公式

\[ \sin{2\theta} = 2\sin{\theta}\cos{\theta},\quad\cos{2\theta} = \cos^2{\theta} - \sin^2{\theta} \]

得到

\[ \sin{\theta} = 2\sin{\tfrac{\theta}{2}}\cos{\tfrac{\theta}{2}},\quad\cos{\theta} = \cos^2{\tfrac{\theta}{2}} - \sin^2{\tfrac{\theta}{2}} \]

先將兩式除以 \(1 = \cos^2{\tfrac{\theta}{2}} + \sin^2{\tfrac{\theta}{2}}\)，再讓分子、分母同除以 \(\cos^2{\tfrac{\theta}{2}}\)，然後代入 \(t = \tan{\frac{\theta}{2}}\)，得到

\[ \sin{\theta} = \frac{2\sin{\tfrac{\theta}{2}}\cos{\tfrac{\theta}{2}}}{\cos^2{\tfrac{\theta}{2}} + \sin^2{\tfrac{\theta}{2}}} = \frac{2\tan{\tfrac{\theta}{2}}}{1 + \tan^2{\tfrac{\theta}{2}}} = \frac{2t}{1+t^2} \] \[ \cos{\theta} = \frac{\cos^2{\tfrac{\theta}{2}} - \sin^2{\tfrac{\theta}{2}}}{\cos^2{\tfrac{\theta}{2}} + \sin^2{\tfrac{\theta}{2}}} = \frac{1 - \tan^2{\tfrac{\theta}{2}}}{1 + \tan^2{\tfrac{\theta}{2}}} = \frac{1-t^2}{1+t^2} \]

由於 \(t = \tan{\frac{\theta}{2}}\)，微分的關係式為

\[\dd{t}=\dv{\theta}\left(\tan{\tfrac{\theta}{2}}\right)\dd{\theta} = \frac{1}{2}\left(1+\tan^2{\frac{\theta}{2}}\right)\dd{\theta}=\frac{1+t^2}{2}\dd{\theta}\]

\(\theta\) 和 \(t\) 的幾何關係可以參考下圖（將 \(\varphi\) 換成 \(\theta\)）。

例題 6-1 ^eq6-1

計算 \(\displaystyle \int{\frac{\dd{\theta}}{1 + \sin{\theta}}}\)。

解.
使用 Weierstrass 代換，設 \(t = \tan{\frac{\theta}{2}}\)，則 \(\sin{\theta} = \frac{2t}{1 + t^2}\)，\(\dd{\theta} = \frac{2 \, \dd{t}}{1 + t^2}\)。積分變為：

\[ \int{\frac{\frac{2}{1 + t^2}}{1 + \frac{2t}{1 + t^2}}\dd{t}} = \int \frac{2 \, \dd{t}}{1 + t^2 + 2t} = \int \frac{2 \, \dd{t}}{(1 + t)^2} \]

進行變數代換，設 \(u = 1 + t\)，則 \(\dd{u} = \dd{t}\)。積分變為：

\[ \int \frac{2 \, \dd{u}}{u^2} = 2 \int u^{-2} \, \dd{u} = -2u^{-1} + C = -\frac{2}{1 + t} + C \]

將 \(t\) 代回，得到：

\[ \int{\frac{\dd{\theta}}{1 + \sin{\theta}}} = -\frac{2}{1 + \tan{\frac{\theta}{2}}} + C \]

例題 6-2 ^eq6-2

計算 \(\displaystyle \int{\csc{x}\dd{x}}\)。

解.

\[ \begin{aligned} \int{ \csc{x}\dd{x} } &= \int{ \frac{\dd{x}}{\sin{x}} } \\ &= \int{ \left(\frac{1 + t^2}{2t}\right) \left(\frac{2}{1 + t^2}\right) \dd{t} }, \quad \text{with } t = \tan{\tfrac{x}{2}} \\ &= \int{ \frac{\dd{t}}{t} } = \ln{|t|}+ C = \ln{\left|\tan{\tfrac{x}{2}} \right|} + C \end{aligned} \]

更多例題：Stewart (9ed), §7.4, Exercises 64–67, p.516

6.2 Euler 代換 ^6-2

Euler 代換（Euler substitution）是針對含有 \(\sqrt{ax^2 + bx + c}\) 的積分，將變數進行代換來簡化積分的方法。 %% 針對 \(\sqrt{ax^2 + bx + c}\)，歐拉提出了三種代換：

  1. 若 \(a > 0\)，設 \(x + \frac{b}{2a} = \sqrt{a} \tan{t}\)。
  2. 若 \(c > 0\)，設 \(x = \sqrt{c} \cot{t}\)。
  3. 若 \(ax^2 + bx + c = 0\) 有實根，設 \(x = u + v\)，其中 \(u\) 和 \(v\) 是方程的兩根。 %% %% #### 例題 6-3 ^eq6-3 計算 \(\displaystyle\int \frac{\dd{x}}{\sqrt{x^2 + 2x + 5}}\)。

解.
使用 Euler 代換法，設 \(x + 1 = 2\tan{t}\)，則 \(\dd{x} = 2 \sec^2{t} \, \dd{t}\)，並且 \(\sqrt{x^2 + 2x + 5} = 2 \sec{t}\)。積分變為：

\[ \int \frac{2 \sec^2{t} \, \dd{t}}{2 \sec{t}} = \int \sec{t} \, \dd{t} = \ln|\sec{t} + \tan{t}| + C。 \]

將 \(t\) 代回，得到：

\[ \int \frac{\dd{x}}{\sqrt{x^2 + 2x + 5}} =\ln \left| \frac{x + 1}{2} + \sqrt{1 + \left(\frac{x + 1}{2}\right)^2} \right| + C。 \]
