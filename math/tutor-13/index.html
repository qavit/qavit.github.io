6. 其他積分技巧 (2) ^6

6.3 使用 Euler 公式積分 ^6-3

使用 Euler 公式積分是將複數指數函數 \(e^{ix}\) 代入三角函數積分中，進而簡化積分的一種方法。

Euler 公式為

\[ e^{ix} = \cos{x} + i \sin{x}。 \]
例題 6-3 ^eq6-3

計算 \(\int e^{ix} \, \dd{x}\)。

解.
使用 Euler 公式，積分變為：

\[ \int e^{ix} \, \dd{x} = \frac{1}{i} e^{ix} + C = -i e^{ix} + C \]

根據 Euler 公式，得到：

\[ -i (\cos{x} + i \sin{x}) + C = -i \cos{x} - \sin{x} + C \]

因此，積分結果為：

\[ \int e^{ix} \, \dd{x} = -i \cos{x} - \sin{x} + C \]
例題 6-4 ^eq6-4

使用 Euler 公式計算積分 \(\displaystyle\int {e^{-x} \cos{x}\dd{x}}\)。

解. 首先，將 \(\cos x\) 表示成

\[ \cos x = \frac{e^{ix} + e^{-ix}}{2} \]

因此，積分可以改寫為

\[ \int {e^{-x} \cos{x}\dd{x}} = \int e^{-x} \left( \frac{e^{ix} + e^{-ix}}{2} \right) \dd{x} = \frac{1}{2} \int \left( e^{(i-1)x} + e^{-(i+1)x} \right) \dd{x} \equiv I \]

於是可以（像實變函數那樣）做積分：

\[ I = \frac{1}{2} \left( \int{e^{(i-1)x} \dd{x}} + \int{e^{-(i+1)x} \dd{x}} \right) = \frac{1}{2} \left( \frac{e^{(i-1)x}}{i-1} + \frac{e^{-(i+1)x}}{-(i+1)} \right) = \frac{1}{2} \left( \frac{e^{(i-1)x}}{i-1} - \frac{e^{-(i+1)x}}{i+1} \right) \]

但是這是複變函數，我們想取出其實部。首先處理分子裡的指數函數，並代入 \(e^{ix} = \cos x + i \sin x\) 和 \(e^{-ix} = \cos x - i \sin x\) ，得到

\[ I = \frac{1}{2} \left( \frac{e^{ix - x}}{i-1} - \frac{e^{-ix - x}}{i+1} \right) = \frac{e^{-x}}{2} \left( \frac{\cos x + i \sin x}{i-1} - \frac{\cos x - i \sin x}{i+1} \right) \]

然後化簡分母：

\[ \frac{1}{i-1} = \frac{1}{i-1}\frac{i+1}{i+1} = \frac{i+1}{-2} \quad \text{和} \quad \frac{1}{i+1} = \frac{1}{i+1}\frac{i-1}{i-1} = \frac{i-1}{-2} \]

代回 \(I\)，得到

\[ \require{color}\require{cancel} I = \frac{e^{-x}}{2}\left[ {\color{blue}\left( \cos x + i \sin x \right) \left( \frac{i+1}{-2} \right)} - {\color{green}\left( \cos x - i \sin x \right) \left( \frac{i-1}{-2} \right)} \right] \]

乘開並化簡：

\[ I = -\frac{e^{-x}}{4} \big[ {\color{blue} (\cancel{i\cos{x}} - \sin{x} + \cos{x} + \cancel{i\sin{x})}} - {\color{green} (\cancel{i\cos{x}} + \sin{x} - \cos{x} + \cancel{i\sin{x})}} \big] \]

於是發現 \(I\) 不含虛部：

\[ I= e^{-x} \left( \frac{-\cos x + \sin x}{2} \right) \]

結論：

\[ \int{e^{-x} \cos{x}\dd{x}} = e^{-x} \left( \frac{-\cos x + \sin x}{2} \right) \]

7. 積分策略 ^7

直到現在，我們已經學會各種積分技巧。但本節我們將隨機介紹一系列雜項積分，主要挑戰在於「辨認應該使用哪種技巧或公式」。==沒有硬性規則可以告訴我們在特定情況下應該使用哪種方法==，但我們提供一些通用指南，這些指南可能會對你有所幫助。

應用這些策略的前提是掌握基本的積分公式。在下表中，我們收集了之前列表中的積分公式以及本章中學到的幾個額外公式。大多數這些公式應該被記住。知道所有這些公式是有益的，但標有星號的公式不需要記住，因為它們很容易推導出來。公式 19 可以通過使用部分分式來避免，而公式 20 可以用三角代換來代替。

一旦掌握了這些基本積分公式，如果你不能立即看出如何攻克給定的積分問題，你可以嘗試以下四步策略：

  1. 盡可能簡化被積函數：有時候使用代數運算或三角恆等式可以簡化被積函數，並使積分方法變得明顯。
  2. 尋找明顯的代換：嘗試在被積函數中找到某個函數 \(u = g(x)\)，其微分 \(\dd{u} = g'(x) \dd{x}\) 也出現，僅差一個常數因子。
  3. 根據形式分類被積函數：如果步驟 1 和 2 沒有解決問題，那麼我們看一下被積函數 \(f(x)\) 的形式。
    • (a) 三角函數：如果 \(f(x)\) 是 \(\sin x\) 和 \(\cos x\)、\(\tan x\) 和 \(\sec x\) 或 \(\cot x\) 和 \(\csc x\) 的乘積，那麼我們使用「[[tutor-11#^1|§三角函數的積分]]」中的代換。
    • (b) 有理函數：如果 \(f(x)\) 是有理函數，我們使用 [[tutor-12#^4|§部分分式法]]。
    • (c) 分部積分：如果 \(f(x)\) 是 \(x\) 的冪次（或多項式）與超越函數（如三角函數、指數函數或對數函數）的乘積，那麼我們嘗試 [[tutor-10#3. 分部積分法 (Integration by Parts)|§分部積分法]]，==按照第 7.1 節中的建議選擇 \(u\) 和 \(\dd{v}\)。如果你看一下第 7.1 節中的函數，你會發現它們大多數是剛才描述的類型==。
    • (d) 根式：當某些根式出現時，建議使用特定的代換。
      • (i) 當 \(\sqrt{x^2 \pm a^2}\) 或 \(\sqrt{a^2 - x^2}\) 出現時，我們根據[[tutor-11#^tri-sub|表格]]使用[[tutor-11#^2|三角代換法]]。
      • (ii) 當 \(\sqrt[n]{ax + b}\) 出現時，我們使用有理化代換 \(u = \sqrt[n]{ax + b}\) → [[tutor-14#^6-4|§根式的處理技巧]]。更一般地說，有時這對 \(\sqrt[n]{g(x)}\) 也有效。
  4. 再試一次：如果前三步沒有產生答案，記住基本上只有兩種積分方法：[[tutor-10#^2|§變數代換法]]和 [[tutor-10#3. 分部積分法 (Integration by Parts)|§分部積分法]]。
    • (a) 嘗試代換法：即使沒有明顯的代換（步驟 2），一些靈感或創意（甚至絕望）可能會提示適當的代換。
    • (b) 嘗試分部積分：雖然分部積分法通常用於步驟 3(c) 中描述的乘積形式，但有時對單一函數也有效。我們會發現它對 \(\tan^{-1} x\)、\(\sin^{-1} x\) 和 \(\ln x\) 有效，這些都是反函數。
    • (c) 操作被積函數：代數運算（如有理化分母或使用三角恆等式）可能有助於將積分轉換為更簡單的形式。這些操作可能比步驟 1 中更為複雜，並可能需要一些創意。
    • (d) 將問題與之前的問題聯繫起來：當你積累了一些積分經驗，你可能會在給定積分中使用類似於之前積分的方法。或者你甚至可能能夠將給定積分表示為之前積分的形式。
    • (e) 使用多種方法：有時需要兩種或三種方法來評估積分。評估可能涉及幾個連續的不同類型的代換，或者可能將分部積分與一個或多個代換結合。

例題 7-1 ^eq7-1

求 \(\displaystyle \int{\frac{\tan^3{x}}{\cos^3{x}}\dd{x}}\)。

解 1.

\[\int{\frac{\tan^3{x}}{\cos^3{x}}\dd{x}} = \int{\tan^3{x}\sec^3{x}\dd{x}}\]

解 2.

\[\int{\frac{\tan^3{x}}{\cos^3{x}}\dd{x}}=\int{\frac{\sin^3{x}}{\cos^6{x}}\dd{x}}=\int{\frac{1-\cos^2{x}}{\cos^6{x}}\sin{x}\dd{x}}\]

例題 7-2 ^eq7-2

求 \(\displaystyle \int{\sin{\sqrt{x}}\dd{x}}\)。

解. 令 \(u=\sqrt{x}\) ，則 \(\dd{x} = 2u\dd{u}\)。

\[\int{\sin{\sqrt{x}}\dd{x}}=2\int{u\sin{u}\dd{u}}\]

8.1 積分表 ^8-1

積分表（table of integrals）是一種的參考資料，列出常見函數的不定積分，使用積分表可以快速查找某些函數的不定積分，省去繁瑣的計算過程。對於一些複雜的積分問題，我們可以利用計算機或數學軟體來求解。

8.2 計算機代數系統 ^8-2

以下是幾種主要的計算機代數系統（CAS）及其功能介紹，並通過例子說明它們的應用。最後，我們將這些系統的特點做一個比較表。

  1. WolframAlpha / Mathematica

    • 功能：WolframAlpha 是一個線上計算工具，而 Mathematica 是 Wolfram 公司開發的桌面版軟體。它們都具有強大的符號計算功能，包括代數、微積分、矩陣運算、數值計算等。
    • 例子：求解定積分 \(\int e^{x^2} \, \dd{x}\)。
      • 在 WolframAlpha 中輸入：integrate e^(x^2) dx 即可得到結果。

  2. Maple

    • 功能：Maple 是由 Maplesoft 公司開發的計算機代數系統，擅長符號計算、數值計算、圖形繪製和程式設計。
    • 例子：求解方程 \(x^3 - 3x + 1 = 0\)。
      • 在 Maple 中輸入：solve(x^3 - 3*x + 1 = 0, x) 即可得到解。

  3. MATLAB

    • 功能：MATLAB 是由 MathWorks 公司開發的高效能語言，主要用於數值計算。通過附加的 Symbolic Math Toolbox，可以進行符號計算。
    • 替代品：[[Octave]]
    • 例子：計算矩陣的逆矩陣。
      • 在 MATLAB 中輸入：inv([1, 2; 3, 4]) 即可得到逆矩陣。

  4. SageMath

    • 功能：SageMath 是一個開源的數學軟體系統，整合了許多數學軟體套件，具有符號計算、數值計算和圖形繪製功能。
    • 例子：計算 \(\int x \sin(x) \, \dd{x}\)。
      • 在 SageMath 中輸入：integrate(x*sin(x), x) 即可得到結果。

  5. SymPy

    • 功能：SymPy 是一個基於 Python 的符號數學函式庫，適用於符號計算和代數操作。它可以與 Jupyter Notebook 整合，提供互動式計算環境。
    • 例子：求導數 \(\dv{x}(\sin(x) \cos(x))\)。
      • 在 SymPy 中輸入： 
        from sympy import symbols, sin, cos, diff
x = symbols('x')
diff(sin(x) * cos(x), x)
        即可得到結果。

9. 數值積分簡介 ^9

數值積分是通過數值方法來近似計算定積分的方法。常見的數值積分方法包括矩形法、梯形法和辛普森法。

方法

矩形法 將積分區間分成 \(n\) 個等長小區間，每個小區間上取矩形的高為函數值，然後求這些矩形的面積和來近似定積分。

梯形法 將積分區間分成 \(n\) 個等長小區間，用梯形面積來近似每個小區間的面積，然後求和來近似定積分。

辛普森法 將積分區間分成 \(n\) 個等長小區間，用二次多項式來近似每個小區間的函數值，然後求和來近似定積分。
