向量函數的微分

3.4 法向量和次法向量

單位法向量（normal vector） \(\mathbf{N}(t)\) 定義為：

\[ \mathbf{N}(t) = \frac{\mathbf{T}'(t)}{\left| \mathbf{T}'(t) \right|} \]

單位次法向量（binormal vector） \(\mathbf{B}(t)\) 定義為：

\[ \mathbf{B}(t) = \mathbf{T}(t) \times \mathbf{N}(t) \]

3.5 定義扭率

扭率（torsion） \(\tau\) 是描述曲線空間扭曲程度的量度。定義為：

\[ \tau = -\frac{\mathbf{B}'(t) \cdot \mathbf{N}(t)}{\left|\mathbf{r}'(t)\right|} \]

多重積分

1. 矩形區域上的二重積分

回憶 \(\mathbb{R}\) 上的定積分（黎曼積分）

\[ \int_{a}^{b}{f(x)\dd{x}} = \lim_{n\to\infty}{\sum_{i=1}^n{f(x_i^{*})\Delta x}} \]

如何拓展到 \(\mathbb{R}^2\)？

假設 \(f:R \subseteq \mathbb{R}^{2}\to \mathbb{R}\) 是二變數函數；其中 \(R\) 是一個閉矩形（closed rectangle），定義為 [ R = [a,b] \times [c,d] = \left{ (x,y) \in \mathbb{R}^{2} \mid a \leq x \leq b, c \leq y \leq d \right}

] 也就是閉區間 \([a,b]\) 和 \([c,d]\) 的笛卡兒積，其中 \(a,b,c,d\in \mathbb{R}\)。

回憶 \(f\) 的圖形（graph）就是點集合

\[ \left\{ (x,y,z) \in \mathbb{R}^{3}\mid z= f(x,y),\ (x,y) \in R \right\} \]

現在定義點集合 \(S\) 是「\(f\) 的圖形底下、\(R\) 上方」的形體（solid）

\[ S = \left\{ (x,y,z) \in \mathbb{R}^{3} \mid 0 \leq z \leq f(x,y), \, (x,y)\in R \right\} \]

我們的目標是找到 \(S\) 的體積（volume）。 ![[tutor18-solid.png|400]] 第一步：將 \(R\) 切成許多個小矩形

\[ R_{ij} = [x_{i-1},x_{i}] \times [y_{j-1}, y_{j}] = \left\{ (x,y) \in \mathbb{R}^{2} \mid x_{i-1} \leq x \leq x_{i},\,y_{j-1} \leq y \leq y_j \right\} \]

每一個小矩形的面積都是 \(\Delta A = \Delta x \Delta y\) 。 ![[tutor18-subrectangles.png|700]]

在小矩形 \(R_{ij}\) 中取一個樣本點（sample point）\((x_{ij}^{*},y_{ij}^{*})\in R_{ij}\) 。 然後用一個長方體的箱子（或柱子）近似 \(S\) 在 \(R_{ij}\) 上方的部分。

  • 長方體的底是 \(R_{ij}\)，面積為 \(\Delta A = \Delta x \Delta y\)
  • 長方體的高是 \(f \left( x_{ij}^{*},y_{ij}^{*} \right)\)
  • 長方體的體積是 \(f \left( x_{ij}^{*},y_{ij}^{*} \right) \Delta x \Delta y\)

所以，眾多長方體體積的和可以用來近似 \(S\) 的體積 \(V\)：

\[ V \approx \sum_{i=1}^{m}\sum_{j=1}^{n}{f \left( x_{ij}^{*},y_{ij}^{*} \right) \Delta x \Delta y} \tag{1} \]

這是一個雙重黎曼合（double Riemann sum）。

假如我們把矩形切分得無窮細（\(m\)、\(n\) 的數量無窮多），可以預期

\[ V = \lim_{m,n\to \infty}\sum_{i=1}^{m}\sum_{j=1}^{n}{f \left( x_{ij}^{*},y_{ij}^{*} \right) \Delta x \Delta y} \tag{2} \]

定義：矩形區域上的二重積分

二重積分（double integral）是積分的一種，計算函數在二維區域上的總和。對於函數 \(f(x, y)\)，如果上面的極限 (2) 存在，其在矩形區域 \(R = [a, b] \times [c, d]\) 上的二重積分存在，或者稱 \(f(x, y)\) 在 \(R\) 上可積，記作

\[ \iint_R f(x, y) \dd{x} \dd{y} = \lim_{m,n\to \infty}\sum_{i=1}^{m}\sum_{j=1}^{n}{f \left( x_{ij}^{*},y_{ij}^{*} \right) \Delta x \Delta y} \tag{3} \]

二重積分的中點法則

中點法則（The Midpoint Rule）是一種數值積分方法，它是把雙重黎曼和中（式 (1)）的樣本點 \((x_{ij}^{*},y_{ij}^{*})\) 指定為小矩形的中心點 \(\left( \bar{x}_i, \bar{y}_j\right)\) ，然後求和，得到二重積分的近似。

\[ \iint_R f(x, y) \dd{x} \dd{y} \approx \sum_{i=1}^{m} \sum_{j=1}^{n}{ f(\bar{x}_i, \bar{y}_j) \Delta x \Delta y}\tag{4} \]

其中 \(\bar{x}_i\) 和 \(\bar{y}_j\) 是小矩形 \(R_{ij}\) 的中心點座標，亦即 \(\bar{x}_{i} = (x_{i} + x_{i-1})/2\)，\(\bar{y}_{j} = (y_{j} + y_{j-1})/2\)。

迭代積分

有一種更方便的方法可以用來計算二重積分 \(\iint_{R}{f(x,y)\dd{x}\dd{y}}\)，特別是在變數可以分離的情況下。

首先定義 \(\int_c^d f(x, y) \dd{y}\) 記號表示「\(x\) 值保持固定，進行 \(f(x,y)\) 從 \(y=c\) 到 \(y=d\) 的積分」。這個過程又叫「\(f(x,y)\) 對於 \(y\) 的偏積分」（partial integration with respect to \(y\)）。

既然 \(\int_c^d f(x, y) \dd{y}\) 的值取決於 \(x\) ，所以它是一個 \(x\) 的函數，可以定義成

\[A(x) = \int_c^d f(x, y) \dd{y}\]

然後對 \(A(x)\) 從 \(x=a\) 到 \(x=b\) 的積分，也就是

\[ \int_{a}^{b} \int_{c}^{d} f(x, y) \dd{y} \dd{x} = \int_{a}^{b} \left( \int_{c}^{d} f(x, y) \dd{y} \right) \dd{x} \tag{3} \]

右式就是迭代積分（iterated integrals），是將二重積分分解成兩個一重積分的過程。

例題： 計算 \(f(x, y) = x^2 + y^2\) 在矩形區域 \(R = [0, 1] \times [0, 2]\) 上的二重積分。

解：

\[ \iint_R (x^2 + y^2) \dd{x} \dd{y} = \int_0^1 \int_0^2 (x^2 + y^2) \dd{y} \dd{x} \]

首先，對 \(y\) 積分：

\[ \int_0^2 (x^2 + y^2) \dd{y} = x^2 y + \frac{y^3}{3} \bigg|_0^2 = 2x^2 + \frac{8}{3} \]

然後，對 \(x\) 積分：

\[ \int_0^1 \left(2x^2 + \frac{8}{3}\right) \dd{x} = \frac{2}{3} x^3 + \frac{8}{3} x \bigg|_0^1 = \frac{10}{3} \]

因此，二重積分的值為 \(\frac{10}{3}\)。

例題： 計算

  • (a) \(\int_{0}^{3}{\int_{1}^{2}{x^{2}y \dd{y} \dd{x}}}\)
  • (b) \(\int_{1}^{2}{\int_{0}^{3}{x^{2}y \dd{x}\dd{y}}}\)

富比尼定理

富比尼定理（Fubini’s Theorem）說明在某些條件下，二重積分的順序可以交換。

如果 \(f\) 在矩形 \(R = [a,b] \times [c,d] = \left\{ (x,y) \in \mathbb{R}^{2} \mid a \leq x \leq b, c \leq y \leq d \right\}\) 上連續，則

\[ \iint_R f(x, y) \dd{x} \dd{y} = \int_c^d \left( \int_a^b f(x, y) \dd{x} \right) \dd{y} = \int_a^b \left( \int_c^d f(x, y) \dd{y} \right) \dd{x} \]

此定理適用於被積函數在積分區域上連續或可積的情況。一般來說，Fubini 定理成立的條件如下：

  • \(f\) 在 \(R\) 上有界，
  • \(f\) 只在「有限數量個光滑曲線上」不連續，
  • 迭代積分存在。

Fubini 定理的證明非常複雜

二重積分於一般區域

定義：一般區域的二重積分

在一般區域 \(D\) 上，二重積分記作：

\[ \iint_D f(x, y) \dd{x} \dd{y} \]

其計算方法與矩形區域類似，但需根據區域的形狀設定積分的上下限。

例題： 計算 \(f(x, y) = x + y\) 在三角形區域 \(D\)（由 \(x=0\), \(y=0\), \(x+y=1\) 所圍成）上的二重積分。

解：

\[ \iint_D (x + y) \dd{x} \dd{y} = \int_0^1 \int_0^{1-x} (x + y) \dd{y} \dd{x} \]

首先，對 \(y\) 積分：

\[ \int_0^{1-x} (x + y) \dd{y} = xy + \frac{y^2}{2} \bigg|_0^{1-x} = x(1-x) + \frac{(1-x)^2}{2} \]

化簡得到：

\[ x(1-x) + \frac{(1-x)^2}{2} = x - x^2 + \frac{1}{2} - x + \frac{x^2}{2} = \frac{1}{2} - \frac{x^2}{2} \]

然後，對 \(x\) 積分：

\[ \int_0^1 \left(\frac{1}{2} - \frac{x^2}{2}\right) \dd{x} = \frac{1}{2}x - \frac{x^3}{6} \bigg|_0^1 = \frac{1}{2} - \frac{1}{6} = \frac{1}{3} \]

因此，二重積分的值為 \(\frac{1}{3}\)。

改變積分順序

在某些情況下，改變積分順序可以使計算更簡單。需要根據區域的形狀重新設定積分的上下限。

二重積分的性質

二重積分具有線性性、加法性和可逆性等性質。例如，對於常數 \(c\) 和函數 \(f(x, y)\)：

\[ \iint_D cf(x, y) \dd{x} \dd{y} = c \iint_D f(x, y) \dd{x} \dd{y} \]
