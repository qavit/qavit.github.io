弗里德曼方程式

3.1 簡介

我們心中的目標是了解宇宙的演化。為了描述宇宙的過去、現在和未來及其動力學，我們需要知道標度因子如何依賴於時間——即 \(R(t)\)。為此，我們必須選擇一個理論模型。標準現代相對論宇宙學的基本成分是：

  1. 愛因斯坦場方程式
  2. 宇宙學原理
  3. 狀態方程式

首先，我們在廣義相對論的框架下假設愛因斯坦場方程式來描述時空在宇宙場的存在下是如何變化的：

\[ G_{\mu \nu} - \Lambda g_{\mu \nu} = R_{\mu \nu} - \frac{1}{2} g_{\mu \nu} R - \Lambda g_{\mu \nu} = \frac{8\pi G}{c^{4}} T_{\mu \nu} \tag{3.1} \]

其中包含：

  • 愛因斯坦張量  \(G_{\mu \nu}\)
  • 度量張量  \(g_{\mu\nu}\)
  • Ricci 張量  \(R_{\mu\nu}\)
  • Ricci 純量  \(R\)
  • 能量應力張量  \(T_{\mu \nu}\)
  • 宇宙學常數 \(\Lambda\)

然後，我們假設宇宙是各向同性的和均勻的，所以宇宙學原理成立。

這樣的粒子分佈（例如星系分佈）顯得像一種完美的、無摩擦的流體。所以我們可以擬定一個狀態方程式。

在我們開始廣義相對論的論述之前，值得一提的是，只用牛頓式表述（Newtonian formalism）就可以計算出宇宙的膨脹和不同的世界模型。使用膨脹球體模型，可以推導出一個類似於所謂的弗里德曼方程式（Friedmann equation）的方程式。這在附錄 A 中完成。牛頓動力學可以幫助我們理解宇宙演化的某些方面。但是，牛頓式模型存在嚴重的局限性：

  • 模型不再是各向同性的，因為所選球體有一個中心點。根據宇宙學原理，不應該有特定一個「朝向中心」的方向。
  • 嵌入宇宙中的球體（a sphere embedded in the Universe）的概念與均勻性以及宇宙學原理相悖。宇宙中沒有特殊的地方可以放置這個球體。
  • 牛頓方法僅使用物質的密度，根據廣義相對論，能量等同於質量，因此，還需要進行一些概括，以便將能量密度考慮進去。

為了解決這些問題，首先必須引入一種特殊的度規，這種度規要符合宇宙學原理。然後，為了確定宇宙的動力學，我們必須了解度規如何因應宇宙中的場（輻射場、物質場等）而隨時間演變。這可以藉助愛因斯坦場方程式 (3.1) 式來完成，其中能量應力張量張量   包含了造成時空曲率的場。

3.2 幾何學與度規

哪一種度規描述了均勻且各向同性的四維時空連續體？其中，四維是三個空間坐標和一個時間坐標的組合？因為四維和三維空間的可視化非常困難，所以我們通常從二維空間開始發展一些想法，這有助於理解三維案例。

二維空間

各向同性而均勻的二維空間的可能性如圖 3.1 所示。右側表格中最後一列的數字給出了三角形的內角和。

請注意，度規描述了空間的局部幾何形狀，並告訴我們如何使用度規張量來計算兩點之間的距離。讓我們考慮最簡單的二維情況—— \(x\)-\(y\) 平面，其中距離由下式給出（使用愛因斯坦的求和約定）：

\[ \require{physics} \dd{s}^{2} = \dd{x_{1}^{2}}+ \dd{x_{2}^{2}} = \sum_{\mu} \sum_{\nu} g_{\mu \nu}\dd{x^{\mu}\dd{x^{\nu}}} = g_{\mu \nu}\dd{x^{\mu}\dd{x^{\nu}}} \tag{3.2} \]

所以我們有 \(g_{11} = g_{22} = 1\)，\(g_{12}=g_{21}=0\)，或

\[ g_{\mu \nu} = \begin{pmatrix}1&0\\0&1\end{pmatrix} \tag{3.3} \]

如果我們使用不同的坐標系，例如 \(x_1 = r'\cos{\theta}\)，\(x_2 = r'\sin{\theta}\)  和 \(r'=Rr\) 的極坐標，如圖 3.2 所示，距離將由下式給出：

\[ \dd{s}^{2} = \dd{r'}^{2} + {r'}^{2} \dd{\theta'}^{2} = R^{2}\left( \dd{r}^{2} + {r}^{2} \dd{\theta}^{2} \right) \tag{3.4} \]

這可以推廣到球體和雙曲面，見圖 3.2。通過一些繁瑣的幾何（留作習題）我們可以獲得

\[ \mbox{球體：}\quad\dd{s}^{2} = R^{2}\left( \frac{{\dd{r}^{2}}}{{1-r^{2}}} + {r}^{2} \dd{\theta}^{2} \right) \tag{3.5} \]\[ \mbox{雙曲面：}\quad\dd{s}^{2} = R^{2}\left( \frac{{\dd{r}^{2}}}{{1+r^{2}}} + {r}^{2} \dd{\theta}^{2} \right) \tag{3.6} \]

極坐標中，這些距離關係都可以通過使用一個新參數 \(k\) 來概括：

\[ \dd{s}^{2} = R^{2}\left( \frac{{\dd{r}^{2}}}{{1-kr^{2}}} + {r}^{2} \dd{\theta}^{2} \right) \tag{3.7} \]

\(k\) 的不同值描述了上面提及的各種二維空間——

  • \(k = 1\) 表示二維球
  • \(k = 2\) 表示平面
  • \(k = -1\) 表示雙曲曲面

再次作為練習，你可以計算球坐標中的空間。結果將是：

\[ \dd{s}^{2} = R^{2}\left( \dd{\theta}^{2} + S_{k}^{2}(\theta)\dd{\phi}^{2} \right) \tag{3.9} \]\[ S_{k}(\theta) = \begin{cases} \sin{\theta}, & k = +1, & \mbox{球面} \\ \theta, & k = 0, & \mbox{平面}\\ \sinh{\theta}, & k = -1, & \mbox{雙曲面} \end{cases} \]

3.2.2 Robertson–Walker 度規

要了解獲得宇宙演化方程式所需的度規標準，我們從兩個已知的要素開始：宇宙學原理和膨脹。讓我們從考慮通常的平直歐幾里得三維空間開始。它由坐標 \((x,y,z)\) 或球坐標 \((r,\theta,\phi)\) 所描述。對於這個（平直、均勻、各向同性的）空間，線元素距離的平方由下式給出：

\[ \dd{s}^{2} = R^{2}\left( \dd{r}^{2} + {r}^{2} \dd{\Omega}^{2} \right) \quad \mbox{及} \quad \dd{\Omega}^{2} = \dd{\theta}^{2} + \sin^{2}\!{\theta}\;\dd{\phi}^{2} \tag{3.9} \]

第一個一般化：可以證明，每個各向同性和均勻的三維空間都可以使用球坐標進行參數化，以獲得：

\[ \dd{s}^{2} = c^{2}\dd{t}^{2} - R^{2}(t)\left( \frac{{\dd{r}^{2}}}{{1-kr^{2}}} + {r}^{2} \dd{\Omega}^{2} \right) \tag{3.10} \]

 \(k\) 的不同值再次描述了不同的空間——

  • \(k = 1\)  表示三維球面
  • \(k = 0\) 表示三維歐幾里德空間
  • \(k = -1\) 表示三維雙曲面。

不過這些空間的可視化並不容易。

第二個一般化：知道宇宙正在膨脹以後，我們現在可以採取另一個簡單的步驟，先假設標度因子 \(R(t)\) ，然後使度規（即三維部分）變得可擴張或收縮。所以要從三維空間到四維空間，可以利用相對論的知識；其中兩個事件之間的距離由

\[ \dd{s}^{2} = c^{2}\dd{t}^{2} - \dd{s_{\rm 3D}^{2}} \]

給出。插入之前（第一個一般化）的關係式，我們得到所謂的Roberson-Walker 度規

\[ \dd{s}^{2} = c^{2}\dd{t}^{2} - R^{2}(t)\left( \frac{{\dd{r}^{2}}}{{1-kr^{2}}} + {r}^{2} \dd{\theta}^{2} + r^{2} \sin^{2}\!{\theta}\, \dd{\phi}^{2} \right) \tag{3.11} \]

它描述了==隨時間變化的一般的、均勻的、各向同性的時空（ the generic, homogeneous, isotropic space-time that changes with time）==。

因此，Roberson-Walker 度規遵循宇宙學原理。寫成 (3.11) 形式的方程式包含了我們宇宙的一個重要屬性——膨脹。透過時間相關的標度因子 \(R(t)\)，宇宙的膨脹也在考慮之內。 我們應該觀察到：

  1. 標度因子 \(R(t)\) 將由物理學決定。換句話說，我們將使用愛因斯坦場方程式獲得標度的時間演化，從而獲得宇宙的時間演化，其中包括改變時空的物質和能量。
  2. 坐標 \((r,\theta,\phi)\) 是共動坐標。\(\theta\)、\(\phi\) 描述了任意觀察者所看到的天空中的方向，我們選擇在地球上的觀察者。請記住，只有對於那些視宇宙為各向同性的觀察者而言，我們才能將共動坐標被分配給他們。它們隨著哈伯流（Hubble flow）移動。這些觀察者會認為 CMB 是各向同性的。請注意，一般來說，本地觀察者相對於共動觀察者具有特定的速度。（譯注：指的可能是地球相對於太陽、太陽相對於銀河系、以及銀河系相對於本星系群的運動）當宇宙膨脹時，共動坐標不會改變。位於 \((r,\theta,\phi)\) 處的星系將保留在那裡，但是，由於標度因子，星系之間的距離發生了變化。
  3. 上式中，小寫的 \(r\) 為徑向坐標。我們必須乘以 \(R(t)\)  才能得到「原距」（proper distance）。換句話說，如果 \(d\) 是兩個星系之間的共動距離，那麼原距應該為 \(D(t)=R(t)d\)，今日的原距離為 \(D(t_0)=R(t_0)d\) 。
  4. 常數 \(k\) 決定了給定時間   時三維超曲面的幾何形狀：
    • \(k = 1\) 為正曲率、曲率半徑為 \(R(t)\)  的超曲面
    • \(k = 0\) 為零曲率、「平直」的超曲面
    • \(k = -1\) 為負曲率、曲率半徑為 \(R(t)\)  的超曲面
  5. 請注意，我們在上面寫的度規留下了歸一化的可能性，可以用以重新調整數量。我們之所以只使用 \(k=-1,0,1\) 這些值，是因為所有其他值都可以通過重新調整 \(R\) 或 \(r\) 找到。例如當  \(k = -5\) 時，我們就使用 \(r'=\sqrt{5}r\) 和 \(R'=R/\sqrt{5}\) ，最後還是會獲得相同形式的 \(k=-1\)。
  6. 時間坐標 \(t\) 是宇宙時間（cosmic time）。不同遙遠星系中的觀測者測量到的是相同的時間，而且他們應該將時鐘同步到這個時間。為此，他們可以使用像是 CMB 這樣的可觀測量。事實上，CMB 的溫度自發射以來一直嚴格遞減。因此，確定了溫度就等於確定了時間。
